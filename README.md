Style Hub is a full-stack e-commerce web application built using Flask.
The project focuses on creating a modern online shopping experience with both frontend design and backend functionality.

## 🚀 Features

* User Authentication System
* Admin Dashboard
* Product Management
* Shopping Cart Functionality
* Order Management System
* Razorpay Payment Integration
* Responsive Modern UI
* Product Image Uploads
* Order Status Management

## 🛠️ Tech Stack

### Frontend

* HTML
* CSS
* Bootstrap
* JavaScript

### Backend

* Python
* Flask

### Database

* SQLite

## 📌 Project Highlights

* Built complete authentication workflow
* Implemented dynamic product management system
* Integrated online payment gateway using Razorpay
* Designed responsive UI for better user experience
* Developed admin panel for managing products and orders
